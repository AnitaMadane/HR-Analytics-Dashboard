# HR Analytics Dashboard (Power BI)

## 📊 Overview
This dashboard analyzes workforce demographics, attrition trends, and salary insights using HR data.

## 🗂 Dataset
- Source: HR_Data.xlsx (1000+ employee records)
- Fields: EmployeeID, Department, Gender, Age, Salary, Job Role, Performance Rating, Work Life Balance, Attrition

## 📈 Dashboard Features
- Total Employees KPI
- Attrition Rate % KPI
- Gender Distribution Donut Chart
- Average Monthly Income by Job Role
- Attrition Trend by Years at Company
- Attrition by Work-Life Balance
- Attrition vs Performance Rating
- Interactive Slicers (Department, Job Role, Gender)

## 🎯 Purpose
- Showcase HR analytics using Power BI
- Provide unique insights (salary vs attrition, work-life balance analysis)
- Build recruiter‑friendly portfolio project
